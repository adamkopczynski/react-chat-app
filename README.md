This project is based on FreeCodeCamp example.

## Installation

  You can try it on your own.

  ```shell
    git clone https://github.com/adamkopczynski/react-chat-app.git

    cd react-chat-app

    npm install
    
    npm start
  ```
  Start node server:
  ```shell
     cd server
     
     node app.js
  ```
